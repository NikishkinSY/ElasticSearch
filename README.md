# ElasticSearch

- .Net Microservices (.Net Core 3.1 edition) (https://docs.microsoft.com/en-us/dotnet/architecture/microservices/)

- Jason Taylor Clean Architecture (https://jasontaylor.dev/)

- .Net Core 3.1 

- ElasticSearch 7.4 (NEST)

- AWS

- Auth0

- Swagger

- Serilog -> ELK

- Docker support
